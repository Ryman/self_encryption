A file content self encryptor

This library will provide convergent encryption on file based data and produce a DataMap type and several chunks of data. Each chunk is max 1Mb in size and has a name. Thsi name is the Sah512` of the content. This allows the chunks to be confirmed and if using size and Hash checks then there is a high degree of certainty in the data validity.
